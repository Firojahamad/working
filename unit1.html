Unit 1: Introduction to Java

The History of Java

Java is a programming language that was first released by Sun Microsystems in 1995. It was designed by James Gosling and his team with the intention of creating a language that could run on any device, known as the "Write Once, Run Anywhere" (WORA) principle.

Key points in the history of Java include:

  • 1991: The project that would become Java started under the name "Oak."
  • 1995: The official release of Java 1.0 by Sun Microsystems.
  • 1996: The introduction of the Java Applet for web browsers.
  • 1998: The release of Java 2 (J2SE 1.2) with significant improvements.
  • 2004: Java underwent a major overhaul with the release of Java 5 (J2SE 5.0), introducing features like generics and metadata annotations.
  • 2010: Oracle Corporation acquired Sun Microsystems, becoming the new steward of Java.
  • 2014: The release of Java 8 brought lambdas and the Stream API.
  • 2017: Java 9 introduced the module system.
  • 2020: The latest LTS (Long-Term Support) version, Java 11, is widely used in enterprise environments.

Java's versatility, portability, and backward compatibility have contributed to its continued popularity in various domains, including web development, mobile applications, and enterprise solutions.
