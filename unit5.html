Unit 5: Arrays and ArrayLists

Arrays

An array in Java is a data structure that allows you to store multiple values of the same type under a single variable name. The elements in an array can be accessed using an index.

// Declaration and initialization of an array
int[] numbers = {1, 2, 3, 4, 5};
// Accessing elements of the array
int thirdElement = numbers[2]; // Value: 3
// Modifying elements of the array
numbers[1] = 10; // Array becomes: {1, 10, 3, 4, 5}

ArrayLists

ArrayList is a part of the Java Collection Framework and provides dynamic resizing, which means you can add or remove elements at runtime. It is part of the `java.util` package.

// Import the ArrayList class
import java.util.ArrayList;
// Declaration and initialization of an ArrayList
ArrayList colors = new ArrayList<>();
// Adding elements to the ArrayList
colors.add("Red");
colors.add("Green");
colors.add("Blue");
// Accessing elements of the ArrayList
String secondColor = colors.get(1); // Value: "Green"
// Modifying elements of the ArrayList
colors.set(0, "Yellow"); // ArrayList becomes: {"Yellow", "Green", "Blue"}
