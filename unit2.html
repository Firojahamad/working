Unit 2: Variables and Data Types

Variables

In Java, a variable is a container that holds data that can be manipulated during the program's execution. Variables must be declared with a specific data type, and they can be assigned values that match the declared type.

Example:

int age; // Declaration of an integer variable
age = 25; // Assignment of a value to the variable
System.out.println("Age: " + age); // Output: Age: 25

Data Types

Java supports various data types, including:

  • Primitive Data Types:
    • int: Integer data type
    • double: Double-precision floating-point data type
    • char: Character data type
    • boolean: Boolean data type
  • Reference Data Types:
    • String: Represents a sequence of characters
    • Arrays: Holds a collection of elements of the same type

It's essential to choose the appropriate data type based on the type of data you want to store.
