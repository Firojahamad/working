Unit 4: Control Flow Statements

Conditional Statements (if, else if, else)

Conditional statements allow you to control the flow of your program based on certain conditions. The basic structure includes:

if (condition) {
// Code to be executed if the condition is true
} else if (anotherCondition) {
// Code to be executed if anotherCondition is true
} else {
// Code to be executed if no conditions are true
}

Switch Statement

The switch statement allows you to select one of many code blocks to be executed. It's often used as an alternative to long chains of if-else statements.

int day = 3;
switch (day) {
case 1:
System.out.println("Monday");
break;
case 2:
System.out.println("Tuesday");
break;
// Add more cases as needed
default:
System.out.println("Unknown day");
}

Looping Statements (for, while, do-while)

Looping statements allow you to execute a block of code repeatedly.

// Example of a for loop
for (int i = 0; i < 5; i++) {
// Code to be executed in each iteration
}
// Example of a while loop
int count = 0;
while (count < 5) {
// Code to be executed in each iteration
count++;
}
// Example of a do-while loop
int num = 0;
do {
// Code to be executed in each iteration
num++;
} while (num < 5);
