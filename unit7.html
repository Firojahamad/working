Unit 7: Inheritance and Polymorphism

Inheritance

Inheritance is a fundamental concept in Object-Oriented Programming that allows a new class to inherit the characteristics and behaviors of an existing class. The existing class is referred to as the superclass or parent class, and the new class is the subclass or child class.

// Example of a superclass (parent class)
class Animal {
void eat() {
System.out.println("Animal is eating");
}
}
// Example of a subclass (child class) inheriting from Animal
class Dog extends Animal {
void bark() {
System.out.println("Dog is barking");
}
}

Polymorphism

Polymorphism allows objects to be treated as instances of their parent class, promoting flexibility and reusability of code. There are two types of polymorphism: compile-time (method overloading) and runtime (method overriding).

// Example of compile-time polymorphism (method overloading)
class Calculator {
int add(int a, int b) {
return a + b;
}
double add(double a, double b) {
return a + b;
}
}
// Example of runtime polymorphism (method overriding)
class Shape {
void draw() {
System.out.println("Drawing a shape");
}
}
class Circle extends Shape {
void draw() {
System.out.println("Drawing a circle");
}
}
