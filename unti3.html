Unit 3: Operators and Expressions

Operators

Operators in Java are symbols that perform operations on variables and values. Java supports various types of operators:

  • Arithmetic Operators: +, -, *, /, %
  • Relational Operators: ==, !=, >, <, >=, <=
  • Logical Operators: && (and), || (or), ! (not)
  • Assignment Operators: =, +=, -=, *=, /=, %=
  • Increment and Decrement Operators: ++, --

Expressions

An expression is a combination of variables, operators, and values that produces a result when evaluated. Expressions are an essential part of Java programming.

Example:

int x = 5;
int y = 10;
int result = x + y; // Addition
System.out.println("Result: " + result); // Output: Result: 15
